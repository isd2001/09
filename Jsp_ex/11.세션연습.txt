# < UP & DOWN >의 Web 버전 #
 game 이라는 folder를 만드시고..
   /in.jsp  : 시작페이지

   /turn.jsp : 사용자가 숫자입력할수 있게 해둘 페이지

   /result.jsp : 결과페이지
     사용자가 넘긴 숫자랑 맞춰야 되는 숫자를 비교해서 
     정답이 아니라면
       UP or DOWN인지 알려주고 
       다시 turn으로 갈수 있게 링크를 설정해둔 HTML을 전송
     정답이라면
       몇번만에 맞춘건지와 함께 
       in.jsp로 갈수있게 링크 설정해둔 HTML을 전송
